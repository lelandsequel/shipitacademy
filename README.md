# Operator Foundry — Weekend Intensives

**Ship something real in 48 hours.**

Operator Foundry runs weekend intensives for builders who want to go from zero to deployed using AI-native tools. No theory. No fluff. Real products, real demos, real momentum.

## The Weekend Format

- **Saturday 9am–9pm**: Build sessions, tool walkthroughs, async sprints
- **Sunday 9am–6pm**: Polish, deploy, demo day
- **12 spots max** per cohort — small enough to be real, big enough to create energy
